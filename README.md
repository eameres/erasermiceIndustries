# erasermiceIndustries

Simple static website starter for GitHub Pages.

## Local files

- `index.html` - page structure
- `style.css` - site styling
- `script.js` - small client-side behavior
- `.github/workflows/pages.yml` - GitHub Actions deployment workflow

## Publish on GitHub Pages

1. Create a new GitHub repository named `erasermiceIndustries`.
2. Upload or push the contents of this folder to the `main` branch.
3. In GitHub, open **Settings** -> **Pages**.
4. Set **Source** to **GitHub Actions**.
5. Push changes to `main` and GitHub will publish the site.

If this is a user or organization site, rename the repository to `<username>.github.io`.
